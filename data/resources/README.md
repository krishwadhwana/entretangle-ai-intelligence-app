# Resource Library

Use this folder for founder/operator reference material that should inform
product creation, design strategy, and AI visual generation.

Suggested structure:

- `product-creation/` - PDFs, docs, playbooks, formulas, manufacturing notes,
  sourcing references, packaging references, product development guides.
- `ai-visual-creation/` - image-generation guides, prompt references,
  campaign art-direction notes, photography/style references, ad creative docs.

Keep filenames descriptive and stable, for example:

- `product-creation/haircare-formulation-guide-2024.pdf`
- `ai-visual-creation/beauty-ad-visual-prompting-notes.pdf`

These files are not automatically indexed by the app yet. They are a curated
repo-level resource library that can later be connected to the Design Studio,
product strategy, and visual-generation prompts.
